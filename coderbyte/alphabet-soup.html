THE PROBLEM

Have the function AlphabetSoup(str) take the str string parameter being passed and return the string with the letters in alphabetical order (ie. hello becomes ehllo). Assume numbers and punctuation symbols will not be included in the string.

Use the Parameter Testing feature in the box below to test your code with different arguments.

Check the problem here

MY SOLUTION

    function AlphabetSoup(str) {

  • return str.split('').sort().join('')

    }

MY ANALYSIS

This problem may appear to be challenging, but is actually rather quite simple. Luckily, we have the .sort() function, which sorts values in an array. Since this item is in a string, we’ll need to convert it to an array before sorting it, and then convert it back to a string once we are finished.

We start off by taking the string and ‘splitting' it into an array, so that each letter gets separated by quotations

  • return str.split('')

Next, we utilize the sort function. In this case, since we’re dealing with letters, we can use the sort function alone like this: .sort(). However, if we were trying to sort an array with numbers, some of them being greater than single digit numbers, we would need to use .sort(function(a,b){return a-b}), as the sort function converts numbers to strings and sorts them in unicode order. Don’t worry about that for now, but you can read up on how the sort function works over here.

  • .sort()

Now that we have our array sorted to our liking, it’s time to convert it back to a string!

  • .join('')

Presto! Our array is converted back to a string, with the letters not separated by anything. Not too shabby!
