THE PROBLEM

Have the function SimpleAdding(num) add up all the numbers from 1 to num. For the test cases, the parameter num will be any number from 1 to 1000.

Use the Parameter Testing feature in the box below to test your code with different arguments.

Check the problem here

MY SOLUTION

    function SimpleAdding(num) {

  • var result=0;
  • for (i=0; i <= num; i++){
  • result += i
  • }
  • return result

    • }

MY ANALYSIS

This is one of the easier of the easy coderbyte problems. It allows us to utilize the ‘+=’ operator to return our result. Let’s analyze it together:

  • var result=0;

Like most coderbyte problems, we start off by creating a new variable and leaving it blank, eventually tying it to our result.

  • for (i=0; i <= num; i++){

We then set up a for loop, where we start with 1, and go all the way up to the number we type in the “num” field. Notice the “i <= num”. Alternatively, we could have written “i < num+1”, and gotten to the same result.

  • result += i

Here we are telling our new variable “result” to add ‘i’ to it’s value, and update it’s value each time. This is like saying "result = result+i"

  • }
  • return result

Presto! We’re all set :)

It's worth mentioning that we intentionally put our return function outside of our for loop. This way, we return the final result once it is finished with all of the sumation, rather than returning the result each time as it gets another number added to it.
