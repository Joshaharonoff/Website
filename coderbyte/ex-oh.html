THE PROBLEM

Have the function ExOh(str) take the str parameter being passed and return the string true if there is an equal number of x's and o's, otherwise return the string false. Only these two letters will be entered in the string, no punctuation or numbers. For example: if str is "xooxxxxooxo" then the output should return false because there are 6 x's and 5 o's.

Use the Parameter Testing feature in the box below to test your code with different arguments.

Check the problem here

MY SOLUTION

    function ExOh(str) {

  • var x = str.match( /[x]/g ).length;
  • var o = str.match( /[o]/g ).length;
  •  return x === o

    • }

MY ANALYSIS

Dear Regex, have you ever steered us wrong? Having a bit of knowledge, as usual, helps us greatly with putting together our function. We’re essentially creating 2 variable...one to find every instance of ‘x’, one to find every instance of ‘o’. Let’s analyze line by line.

Here we create a variable, x, and tell it to match every instance in our (str) parameter in which the letter ‘x’ is displayed. We then count the amount of times an ‘x’ is found by adding ‘.length’ at the end.

  • var x = str.match( /[x]/g ).length;

Next, we do the same thing, this time searching for every instance of ‘o’ in our (str) parameter.

  • var o = str.match( /[o]/g ).length;

Last but not least, we want to know if they are equal or not...after all..that’s the whole point of our function right? Simple. We create a statement that will evaluate to either “true” or “false” (what’s known as a boolean), by making a statement that the two variable equal each other. Oh and don’t worry, the computer will know if you’re liying!

  •  return x === o
