THE PROBLEM

Have the function FirstFactorial(num) take the num parameter being passed and return the factorial of it (ie. if num = 4, return(4 * 3 * 2 * 1)). For the test cases, the range will be between 1 and 18.

Use the Parameter Testing feature in the box below to test your code with different arguments.

Check the problem here

MY SOLUTION

    function FirstFactorial(num) {

  • var numberArray=[];
  • for(i=1; i <= num; i++){
  • numberArray.push(i)
  • }
  • return numberArray.reduce (function(a,b){
  • return a*b
  • });

    • }

MY ANALYSIS

There are a number of ways one can do this. I chose to utilize the .reduce() function, which keeps things nice and neat in a problem like this. Let’s go over it:

Like most Javascript problems, I first set up a new variable which will be tied to our answer

    var numberArray=[];

Here we set up a for loop, starting with the number 1, and going all the way up until the number itself.

  • for(i=1; i <= num; i++){

We then push each of those numbers to our new array

  • numberArray.push(i)
  • }

Here’s where we spice things up a bit with the .reduce() function. I haven’t wrapped my head on exactly how the .reduce() function works, but my understanding is that it allows you to flatten an array into 1 variable, and determine how that variable will be computed. In this case, we chose to take each variable in the array, and multiply it by itself by the line “return a*b”.

  • return numberArray.reduce (function(a,b){
  • return a*b
  • });

Alternatively, we could have ran a 2nd for loop that ran through the newly created array, and multiplied each number by itself. Try it out - see if you can solve it this way too :)
