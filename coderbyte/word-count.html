THE PROBLEM

Have the function WordCount(str) take the str string parameter being passed and return the number of words the string contains (ie. "Never eat shredded wheat" would return 4). Words will be separated by single spaces.

Use the Parameter Testing feature in the box below to test your code with different arguments.

Check the problem here

MY SOLUTION

    function WordCount(str) {

  •  var numberOfSpaces = str.trim().match( / /g )
  •  if (numberOfSpaces != null) {
  •    var numberOfWords = numberOfSpaces.length + 1
  •   }
  • else { var numberOfWords = 1 }

    • return numberOfWords;

    }

MY ANALYSIS

Once again, our dear friend Regex shows up to save the day

Essentially what we are doing is telling the code to search for spaces in the string in between words (not including spaces before the first word and after the last word). Then count up all the spaces you find, add 1, and give us the number. Since we’re told each word will only be separated by one space, this method should work well.

We start with creating a new variable, “numberOfSpace” which does just that. The first step is for us to use the trim() function, which will cut off any spaces appearing before the first word and any spaces appearing after the last word.

  • str.trim()

We then apply the match function to search for every blank space in our string

  • .match( / /g )

Next, we set an if statement. This is important, because there may just be one word in our string, which would make the value for our numberOfSpaces variable null, throwing off our entire formula.

  •  if (numberOfSpaces != null) {

In the event that there are atleast 2 words in our string, we’re going to assign a new variable, numberOfWords to equal the length of our numberOfSpaces variable, plus one.

  •    var numberOfWords = numberOfSpaces.length + 1
  •   }

Otherwise, if there is just one word in our string, making our numberOfSpaces value equal to null, we will assign a value of 1 to our numberOfWords value.

  • else { var numberOfWords = 1 }

Buy a ribbon, tie a nice bow and wrap it up - now we’re going to return our numberOfWords variable and we’re all set!

    return numberOfWords;

    }
