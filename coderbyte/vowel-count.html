THE PROBLEM

Have the function VowelCount(str) take the str string parameter being passed and return the number of vowels the string contains (ie. "All cows eat grass" would return 5). Do not count y as a vowel for this challenge.

Use the Parameter Testing feature in the box below to test your code with different arguments.

Check the problem here

MY SOLUTION

Best solution:

    function VowelCount(str) {

  •   return (str.match( /[aeiou]/g ).length;

    }

MY ANALYSIS

Before i learned about Regex, i did this problem a different way (see alternative solution below)

I created 2 for loops, where we would take the first character in the first array (which was all vowels), and run through each letter of the string that we input to see if any of the characters matched that first vowel. If so, we would add a value to a new variable, which we named ‘result’.

We then input the values that we want to match, using regex. These values need to be surrounded by dashes /like this/. Note that we also added the global search flag (/g), which will be sure to continue searching the entire string, rather than just stopping at the first match (/[aeiou]/g)
