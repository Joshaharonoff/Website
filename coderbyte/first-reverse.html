THE PROBLEM

Using the JavaScript language, have the function FirstReverse(str) take the str parameter being passed and return the string in reversed order.

Use the Parameter Testing feature in the box below to test your code with different arguments.

Check the problem here

MY SOLUTION

    function FirstReverse(str) {

  • var result='';
  • for(i =str.length-1; i >= 0; i--){
  • result+=str[i]
  • }

    • return result;

    }

MY ANALYSIS

This is challenge is presented first most likely for a reason - it forces you to do one of 2 very important things in javascript:

  • - Navigate a for loop
  • - Switch between a string and an array

In the example I used above, I used a for loop. Let's analyze it together:

I first set up a new variable, "result", where I will be recording the final answer.

  • var result='';

I then set up a for loop, starting from the last letter in the string, and ending with the first letter.

  • for(i =str.length-1; i >= 0; i--){

Finally, I code it so that each letter in the string (str[i]) is being added to the final result.

  • result+=str[i]

and we're done!

    return result;

 

Let’s try it now using a different way, this time converting it to an array, reversing the array with the .reverse() function, and then converting it back to a string:

Looks much simpler no? Let's analyze what we did:

    function FirstReverse(str) {

  • return str.split('').reverse().join('')

    • }

This tells our function to return the following: whatever letters we put inside the parenthesis of our function, converted to an array, and separated by a ‘’ for each letter. If we input “hello”, at this point it should be converted to[ 'h', 'e', 'l', 'l', 'o' ]

  • return str.split('')

This will then reverse the array, so that it now reads [ 'o', 'l', 'l', 'e', 'h' ]

  • .reverse()

Finally, this tells our function to convert these items back to a string, and separate the letters with nothing.

  • .join('')

Great work! A very basic but important set of actions to become familiar with in Javascript.
