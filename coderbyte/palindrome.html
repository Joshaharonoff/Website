THE PROBLEM

Have the function Palindrome(str) take the str parameter being passed and return the string true if the parameter is a palindrome, (the string is the same forward as it is backward) otherwise return the string false. For example: "racecar" is also "racecar" backwards. Punctuation and numbers will not be part of the string.

Use the Parameter Testing feature in the box below to test your code with different arguments.

Check the problem here

MY SOLUTION

    function Palindrome(str) {

  • var forward = (str.match(/[a-z]/g)).join('');
  • return forward === forward.split('').reverse().join('');

    • }

MY ANALYSIS

This problem tests your ability to identify letters only ( as opposed to numbers and spaces), and determine whether it reads the same forward as it does backward. A good function we can use is the reverse() function, which will reverse array items. For that reason, it is also important to be able to convert things easily to an array and a string.

Here we’re creating a new variable, “forward”, which will first search for all letters in the string, and only letters (no number or spaces). This will return our str parameters in an array format.

  • var forward = (str.match(/[a-z]/g))

we then convert it to a string

  • .join('')

Now we need to test if it is the same forward as it is backward. We write a return statement, where if our str parameter is the same forward as it is backwards, it will return true. If it is not the same forward as it is backwards, it will return false. Sounds simple enough, right?

Here we take our forward variable and convert it back to an array

  • return forward === forward.split('')

Then we reverse the array

  • .reverse()

And finally, convert it back to a string. All done!

  • .join('')
