THE PROBLEM

Have the function CheckNums(num1,num2) take both parameters being passed and return the string true if num2 is greater than num1, otherwise return the string false. If the parameter values are equal to each other then return the string -1.

Check the problem here

MY ANALYSIS

This problem is actually the easiest of the easiests in my opinion. It utilizes 1 function - "if".

We have 3 arguements here. Let's analyze each one:

  • if (num2>num1){
  • return true
  • }

Here we're anaylzing both numbers that we are entering in our function. If the 2nd number that we input is greater than the 1st number, return the word "true".

  • else if (num1>num2){
  • return false
  • }

In the event that the 1st number is greater than our 2nd number, return "false"

  • else if (num1===num2){
  • return -1
  • }

And last but not least, if number 1 and number 2 equal each other... return '-1'
